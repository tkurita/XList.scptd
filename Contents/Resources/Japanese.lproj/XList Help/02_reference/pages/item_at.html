item_at

item_at(an_index)

Obtain an item specified with an index number. When a list of indexes is passed as an argument, multiple items will be obtained.

Parameters
  • an_index(integer or list of integer) : an index number or a list of indexes of the items to obtain
Result

anything : Return an_index th item. error number -1728 will be rased when an item can not be obtained.
